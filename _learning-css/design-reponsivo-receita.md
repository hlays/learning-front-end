# Design Responsivo

* Mobile First
* Proporções 
* Usar medidas flexíveis (ex: %)
* Layout Fluído

## Font-size (% e em)
**font-size: 100%;**

=

**font-size: 1em;**

O tamanho da fonte sempre é em relação ao tamanho da fonte pai.

ex:

```
<h1>Fonte 20px<h1>
   <p>
    Fonte 80% ou 0.8em (equivale a 16px -> 20 * 80%)
   </p>
```
